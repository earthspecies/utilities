Howto

  1. Open a file using the 'Open audio file' button.
  2. Specify where to output annotations to using the 'Specify directory ...' button.
  3. Play the recording AND make sure to click somewhere outside the player (for instance here on this howto).
  4. Press spacebar to add annotation(s).
  5. When done, navigate to the directory you specified to locate file with annotations (note: it is compatible with Raven).
